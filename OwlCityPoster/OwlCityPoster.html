Owl City Album Poster

Graphic design for design prompt.


Duration- Feb 2022, 1 week
Type- Solo project
Context- Class assignment
Skills- Graphic Design, Typography, Vector Artwork

About this project

This poster promotes Owl City’s newest album (“Cinematic”). I wanted to portray a fun, whimsical mood similar to the vibe of the music, so I decided to go with fun little illustrations. I also wanted to tease the album’s songs. For both of those reasons, I created vector images, each of which reference a different song in the album.

Iteration

Before finding the final direction, I experimented with different color schemes and compositions, such as using a gradient background to add the color, and using a dark color scheme.
